Molly

A hardened version of Signal

About

Molly is an independent Signal fork for Android

Install

  • Molly F-Droid Repo
  • Github Releases

Versions

Molly, like Signal, uses Google’s proprietary code to support some features.

Molly-FOSS is the community effort to make it 100% free and open-source.

  • Join us at #mollyim:matrix.org on Matrix

Features

Fully FOSS

Contains no proprietary blobs, unlike Signal.

Encrypted

Protects database with passphrase encryption.

Automatic locking

Locks down the app automatically when you are gone for a set period of time.

RAM Shredding

Securely shreds sensitive data from RAM.

Backup scheduling

Automatic backups on a daily or weekly basis.

Tor support

Supports SOCKS proxy and Tor via Orbot.
